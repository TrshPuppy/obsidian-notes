# OPN Attacks
OPN (Open Public Networks) are wifi networks which *are not protected by a password/ PSK*, meaning anyone can connect to them. Additionally, they *lack [encryption](../../OSCP/password-attacks/README.md)* so user traffic can easily be intercepted and read.

Some OPN networks use a [captive portal](captive-portal-bypass.md) to force users to authenticate (usually via a username and password), but there are plenty of ways to get around this (which these notes cover).
