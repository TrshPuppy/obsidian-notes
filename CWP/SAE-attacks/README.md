# SAE Attacks
[SAE](../../networking/wifi/WPA3.md#SAE) is a password-based authentication protocol used in WPA3 networks. It is a very secure protocol, so attacks against these networks mainly rely on misconfigurations and tricking users into authenticating with a rogue AP. There are also some newer DoS techniques like [dragondrain](dragondrain.md) (which just interrupt the network's availability).
